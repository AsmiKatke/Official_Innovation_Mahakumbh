
# 🌟 SNDT Innovation Mahakumbh Official Portal

The official university-wide event website for the flagship **Innovation Mahakumbh** hosted at SNDT University. This portal was architected to handle high-traffic audience engagement, serving as the central hub for event schedules, attendee registrations, and showcasing key innovation highlights.

## ✨ Key Features
* **Responsive User Interface:** Fully optimized for mobile, tablet, and desktop viewing to ensure a seamless user experience for all attendees.
* **Centralized Dashboard:** Organizes dynamic event schedules, speaker line-ups, and innovation showcases.
* **Registration Management:** Streamlined UI for user sign-ups and participation tracking.
* **High Reliability:** Deployed end-to-end via GitHub Pages, maintaining 100% uptime throughout the entire event lifecycle.
